## Log Format

Typical Snort alert format includes:

- Timestamp
- Source IP address
- Destination IP address
- Protocol used
- Alert classification
- Priority level

---

## Purpose of Logs

- Detect suspicious activity
- Identify attacker IP address
- Analyze attack patterns
- Provide evidence for incident reports
